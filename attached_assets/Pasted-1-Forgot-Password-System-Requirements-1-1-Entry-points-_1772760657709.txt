1.1 Entry points (Frontend)

Add links on Login page:

Forgot password?

(Optional) Resend verification code (if MFA/email OTP used)

Flow screens:

Request reset

Check email message

Reset password form

Success → redirect to login

2) Reset Flow (High Security)
Step A — Request Reset

UI

Input: Email

Button: Send Reset Link

Backend
POST /auth/forgot-password
Body:

{ "email": "user@example.com" }

Response (always the same)
Return generic message to prevent email enumeration:

“If an account exists for this email, you’ll receive a reset link.”

Rules

Rate limit: e.g., 5 requests/hour per email + per IP

Log event in audit: PASSWORD_RESET_REQUESTED

Step B — Reset Token Generation

Backend creates a one-time token:

random 32–64 bytes (cryptographically secure)

store hash of token in DB (never raw token)

set expiry: 30 minutes

set used_at when consumed

tie to user + purpose

Token must be invalid if:

expired

already used

user disabled

reset requested again (optional: invalidate old tokens)

Step C — Email Reset Link

Email is sent to user with link:
{frontend_url}/reset-password?token=xxxxx

Must be HTTPS only.

Step D — Reset Password (User sets new password)

Frontend

New password

Confirm password

Backend
POST /auth/reset-password
Body:

{
  "token": "raw_token_from_email",
  "new_password": "NewStrongPassword123!"
}

Backend validation

Validate token (hash match + not used + not expired)

Validate password policy (below)

Update password hash (Argon2/bcrypt)

Mark token used

Invalidate all active sessions/refresh tokens

If MFA enabled: keep MFA; optionally require re-verify after login

Log audit: PASSWORD_RESET_SUCCESS

Response

success message + redirect to login

3) Password Policy (Required)

Enforce on signup + reset + admin reset:

Minimum:

Length >= 12

At least 1 uppercase, 1 lowercase, 1 number

Recommend 1 special character

Block common passwords (recommended)

Security:

Lockout throttling for repeated reset attempts

Password history (optional, recommended): cannot reuse last 3 passwords
