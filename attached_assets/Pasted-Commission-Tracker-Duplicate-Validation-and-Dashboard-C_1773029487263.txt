Commission Tracker Duplicate Validation and Dashboard Count Fix Requirement
1. Same Student Showing Multiple Times in Dashboard

Currently, the same student is appearing multiple times in the dashboard student list even though it looks like the same record.

In the screenshot:

same Agentsic ID is repeated

same Student ID is repeated

same Student Name is repeated

This means duplicate validation is not working correctly, or dashboard is not grouping records properly.

2. Required Duplicate Validation Rule

System must validate and prevent duplicate student creation before saving a new record.

Must check these combinations:
Rule 1

Do not allow save if:

Student Name + Agentsic ID already exists

Rule 2

Do not allow save if:

Student Name + Provider + Student ID already exists

Rule 3

Do not allow save if:

Provider + Student ID already exists

Additional validation

If both below are same, it should also be treated as same student unless business logic says otherwise:

Agentsic ID

Student ID

If duplicate is found, system must block creation and show error message.

3. Error Message Requirement

When duplicate found, show clear validation error.

Example messages:

This student already exists with the same Agentsic ID.

Duplicate student found with same Student ID and Provider.

This student record already exists. Please edit the existing record instead of creating a new one.

Do not allow record to be saved until duplicate issue is resolved.

4. Dashboard Should Not Show Duplicate Student Rows

Dashboard student table should not show multiple rows for the same master student record unless they are intentionally different valid records.

Required logic

Dashboard must use a unique master key, preferably:

Internal ID as primary key

If one student has one master record, dashboard should show that student once only.

Important

If same student has multiple term records:

dashboard should still show one consolidated student row

not one row per term

notes/status can summarize term data inside that single row
