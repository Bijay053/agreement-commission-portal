Contact Edit Feature – Requirement
1. Add Edit Option on Contact Card

Each contact card should have an Edit icon (✏️) beside the Delete icon.

Example layout:

[Contact Name]        ⭐ Primary        ✏️ Edit    🗑 Delete
Position
📧 Email
📞 Phone
Current Issue

Only Delete option exists.

Required

Add:

✏️ Edit

on the top right corner of every contact card.

2. Edit Contact Modal / Form

When user clicks Edit, open a modal popup with editable fields.

Fields
Field	Type	Required
Contact Name	Text	Yes
Position / Role	Text	Optional
Email	Email	Yes
Phone Number	Text	Optional
Primary Contact	Toggle	Optional
3. Primary Contact Logic

Rules:

• Only 1 Primary contact per agreement

If user marks another contact as primary:

Show confirmation popup:

"This will replace the existing primary contact. Continue?"
4. Buttons in Edit Modal
Cancel
Save Changes

Save Changes will:

• Update contact in database
• Refresh contact list without page reload

5. UI Behaviour
Action	Behaviour
Edit	Opens modal
Delete	Confirmation popup
Primary	Blue badge
Update	Instant refresh
6. Delete Confirmation

When clicking delete:

Are you sure you want to delete this contact?
Cancel | Delete

Only users with permission:

Agreement Edit
OR
Admin

• Delete contact
